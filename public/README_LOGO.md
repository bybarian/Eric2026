# Custom Header Logo Instruction

The application's top-left brand header now dynamically supports a custom image! 

## How to Upload

1. Save or rename your custom logo image as **`logo.png`**.
2. Upload this file directly to the root **`public/`** folder of this workspace (so that its path is `/public/logo.png`).
3. Refresh the application.

## Graceful Fallback
If `logo.png` does not exist or fails to load, the header will automatically and seamlessly fall back to the elegant green square containing the letter **`C`** (Cathay General Hospital), keeping the application visually perfect at all times.
